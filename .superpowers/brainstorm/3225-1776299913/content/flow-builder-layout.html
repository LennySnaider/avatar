A — Full Layout

  • Drag & drop desde sidebar (familiar para usuarios de N8N)
  • Properties siempre visible — edición rápida
  • Mismo patrón que AgentSoft (menos código nuevo)

B — Canvas-first

  • Más espacio para el canvas (mejor con flows grandes)
  • UI más limpia y moderna
  • Popover properties = menos distracción
